# Plc Non Visual Components for Delphi 12

A collection of **non-visual VCL components built with Delphi 12 Community Edition (v12.1 Patch 1)** developed to simplify common application tasks and promote reusable, well-structured solutions.

The package is designed for developers who want practical building blocks that integrate cleanly with the Delphi IDE and follow modern coding conventions.

The project is released as **open source under the MIT License**.

---

## 📦 Package Overview

**Package name (palette):** `Plc Non Visual`

This package registers a set of non-visual components that appear in the Delphi component palette and can be dropped onto forms or data modules.

All components are intended to:

* Reduce boilerplate code
* Centralize recurring logic
* Improve maintainability and consistency

---

## 🧩 Included Components

### **TPlFlyMenuManager**

Manages dynamic menu creation based on the contents of a folder, with automatic updates when files change.

### **TPlLanguage**

Provides basic infrastructure for application language management and localization support.

### **TPlLockWndManager**

Helps control window locking and unlocking to prevent user interaction during critical operations.

### **TPlRecentFileComponent**

Implements a standardized and reusable recent-files management system.

### **TPlRunTimeDesigner**

Offers runtime design capabilities, allowing controlled modification of UI elements while the application is running.

### **TPlStylesMenuManager**

Manages VCL styles at runtime, including dynamic menus for selecting and applying styles.

---

## 🛠 Installation

1. Open Delphi 12
2. Add the source files to a package project
3. Build and install the package
4. The components will appear in the **"Plc Non Visual"** palette
